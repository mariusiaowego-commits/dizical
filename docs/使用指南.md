# 🎵 dizical 使用指南

## 基本信息

- **数据库路径**: `~/dev/dizical/data/dizi.db` (课程/缴费), `~/dev/dizical/data/dizical.db` (练习追踪)

- **配置文件**: `.env` (与 dizical 同目录)

## 命令总览

### 课程管理

```bash
dizical lesson generate 2026-05     # 生成5月课程计划
dizical lesson list                  # 本月课程列表
dizical lesson list 2026-05         # 指定月份
dizical lesson calendar              # 日历视图
dizical lesson add 2026-05-03       # 添加课程
dizical lesson cancel 2026-05-03    # 取消课程
dizical lesson reschedule <原日期> <新日期>  # 调课
dizical lesson confirm 2026-05-03    # 确认已上课
```

### 缴费管理

```bash
dizical payment status              # 查看当前应缴状态
dizical payment record 1800         # 记录缴费1800元(现金)
dizical payment history             # 缴费历史
```

### 练习追踪

```bash
dizical practice log 单吐:20        # 打卡（默认昨天）
dizical practice log 单吐:10 --log "突破5连吐"  # 打卡+进展记录
dizical practice week               # 本周日历视图 + 项目分布 + 每日详情
dizical practice dashboard          # 月度热力图 + 项目累计 + 近8周趋势
dizical practice calendar 4         # 4月日历视图
dizical practice stats 4            # 4月统计
dizical practice items             # 练习项目库
dizical practice category list      # 大科目列表
dizical practice category add 气息练习  # 新增大科目
dizical practice category set-item 单吐练习 基本功  # 设置小科目归属
dizical practice config            # 增删改查 TUI（配置管理）
dizical practice import <csv>               # 导入时长 CSV
dizical practice import_logs <csv>          # 批量导入进展 log
dizical practice import-assignments <csv>  # 批量导入每周老师要求
```

### 同步

```bash
dizical reminders sync              # 同步 Apple Reminders
dizical obsidian export 4           # 导出4月报告到 Obsidian
```

### 数据备份

```bash
dizical backup run                  # 执行备份
dizical backup list                 # 查看备份状态
```

## 数据库直接查询

### 方法一：命令行 sqlite3

macOS 自带 SQLite，直接查：

```bash
sqlite3 ~/dev/dizical/data/dizi.db "SELECT * FROM daily_practices ORDER BY date DESC LIMIT 5"
```

带格式输出：
```bash
sqlite3 -header -column ~/dev/dizical/data/dizi.db "SELECT date, total_minutes, items FROM daily_practices ORDER BY date DESC LIMIT 5"
```

常用表：
- `daily_practices` — 每日练习 (date, items JSON, total_minutes, log)
- `lessons` — 课程 (date, time, status, fee, fee_paid)
- `payments` — 缴费 (payment_date, amount, payment_method)

### 方法二：Datasette 网页界面（推荐）

本地开一个网页数据库浏览器，可以写 SQL 也可以点点看看：

```bash
pip install datasette
datasette serve ~/dev/dizical/data/dizi.db --port 8001
```

然后浏览器打开 **http://localhost:8001**

**更方便的方式**，加一个 alias 一步打开：
```bash
# 只需运行一次，以后都有效
echo 'alias dizical-web="cd ~/dev/dizical && datasette serve data/dizi.db --port 8001"' >> ~/.zshrc && source ~/.zshrc

# 以后每次用
dizical-web
```

## 练习打卡格式

### 大科目 / 小科目

dizical 用两级结构管理练习项目：

- **大科目** (category): 气息练习、基本功、歌曲-吹、小节、句子 等
- **小科目** (item): 具体练习内容，属于某个大科目

### 打卡语法

```bash
# 基本打卡
dizical practice log 单吐:15 基本功:10

# 带进展记录
dizical practice log 回娘家:10 --log "找到原因左手放下改善"

# 自动归类（小科目已在大科目下，自动识别）
dizical practice log 全按作低音re:5
```

### CSV 导入格式

**练习时长** (`practice import <csv>`):
```csv
Date,单吐,基本功,采茶扑蝶,回娘家
2026-04-28,8,8,9,3
2026-04-27,10,0,0,3
```

**进展记录** (`practice import_logs <csv>`):
```csv
Date,Log
2026-04-26,今天 基本功，纠正吹口位置效果明显
2026-04-24,找到原因左手放下改善
```

**每周要求** (`practice import-assignments <csv>`):
```csv
WeekStart,Item,Requirement
2026-04-27,长音,多练低音
2026-04-27,回娘家,练了1遍
2026-04-27,唱茉莉花（二）,分析
```
